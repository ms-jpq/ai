# Discover

- Find all source files relevant to the topic:
  - Read every file — don't sample.

  - Include invocations.

- What patterns appear?
  - Rank by prevalence.

  - Highlight uncommon patterns.

# Iterate

- Compare with matching rules file if one exists.
  - Discrepancies.

  - Undocumented patterns.

# Record

- Write `<topic>.md` in the current working directory with each step.

- At end of session, distill `<topic>.md` into rule or skill.
